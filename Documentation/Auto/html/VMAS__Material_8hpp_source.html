21/*
22 VMaterial represents the complete render state configuration necessary for rendering a mesh.
23 This includes shader programs, texture bindings, sampler states, and any relevant GPU state.
24
25 Each mesh rendered in the scene must be associated with a material. VMaterial instances manage
26 the link between shader resources and their required parameters.
27
28 The renderer maintains a library of default materials and templates that can be used as a base
29 for creating customized materials specific to an entity or render pass.
30*/
